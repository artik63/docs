---

Using **`BLOOD-QT-EXTENSION`** allows you to expand **`BLOOD`** functionality and improve your experience working with the bot.

**Key features**

* Perform **swaps** with **`BLOOD`** using popular terminals like **BullX** (PumpVision + token pages) and **Photon** (Memescope + token pages)
* Expand your position management capabilities: view, sort, sell, activate, deactivate, customize strategies and more using our **UI**

_There is no way you can use this product without the actual **`BLOOD`** software (bot)._

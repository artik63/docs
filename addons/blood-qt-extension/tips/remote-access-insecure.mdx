## **Setting up NGROK**

NGROK is a tool that allows you to bring your local server to the internet. Using NGROK, you can run extension on your local machine while the bot runs on your VPS.

## Installation

* Firstly you have to download NGROK **on your VPS** (NOT local) - [https://download.ngrok.com/windows?tab=download](https://download.ngrok.com/windows?tab=download) (macOS and Linux available also)
* You need to sign up on their site so you can receive **auth-token**
* Once downloaded, open **ngrok.exe** and use following command:

```
ngrok config add-authtoken <YourTokenFromDashboard>
```

* After you added token, use `ngrok http 50071` (or +1,2... port if you are using instance 2 / instance 3...) to receive link, paste it in extension, and save. **DON'T close this NGROK window otherwise you WON'T be able to connect (similar to bloodline logic).**

After all those steps you can use extension locally and keep the bot on VPS. There might be different proxying services available but NGROK is easiest way for you to control positions outside of server.

### Protection (optional)

You have ability to use `basic auth` in settings. Generally, you can input name and pass which will be used to gate external requests for your API. Firstly you have to set it in ngrok:

```
ngrok http --basic-auth "name:password" 50071
```

Then you paste both name and password in your extension and ngrok will NOT process any requests with different / without credentials on your server.

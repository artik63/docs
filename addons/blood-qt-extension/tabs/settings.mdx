### Base URL

<Note>

</Note>

Paste here URL where API is served.

## Basic Auth

Optional setting, please check more information in [remote-access-secure.md](/addons/blood-qt-extension/tips/remote-access-secure) or [remote-access-insecure.md](/addons/blood-qt-extension/tips/remote-access-insecure). If you use extension on same machine where's bot located - leave empty.

<Note>
For example, using task with `watch` mode on your `instance 1` means you should paste [http://localhost:50071/](http://localhost:50071/). More information about proxying your requests can be found in [remote-access-secure.md](/addons/blood-qt-extension/tips/remote-access-secure) or [remote-access-insecure.md](/addons/blood-qt-extension/tips/remote-access-insecure).
</Note>

### Default buy amount

Your own `amount of TokenIn`. Leaving field `empty` you will use **exact amount from task.**

| Button   | Action                                                                              | Example                                       |
| -------- | ----------------------------------------------------------------------------------- | --------------------------------------------- |
| Disperse | Select **ONE WALLET** to distribute `FROM` and destination wallets in `TO` section. | [Video](/gui/other-sections/disperse-collect-balance#disperse) |
| Collect  | Select wallets to collect `FROM` and destination **WALLET** in `TO` section.        | [Video](/gui/other-sections/disperse-collect-balance#collect)  |
| Amount   | Specify Amount of tokens to send                                                    |                                               |
| Token    | Specify token address. For SOL transfer you can leave field empty                   |                                               |

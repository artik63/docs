| Button                    | Action                                            |
| ------------------------- | ------------------------------------------------- |
| **5**                     | Sell 5% of your position.                         |
| **25**                    | Sell 25% of your position.                        |
| **50**                    | Sell 50% of your position.                        |
| **100**                   | sell 100% of your position                        |
| **Active**                | Activate previously hidden position.              |
| **Hide**                  | Hide currently active position (reduce RPC load). |
| **Delete**                | Delete position from bot.                         |
| **Refresh positions**     | Refresh value of all your positions.              |
| **Show Hidden positions** | Show previously hidden positions.                 |

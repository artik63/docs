<Note>
GUI is available only on Windows. We highly suggest to use the bot only on server to be able to achieve the best performance
</Note>

<Steps>
<Step title="Download GUI from Dashboard or Discord Bot">
* Dashboard: use [this link](https://dashboard.blood-solutions.com/) to download
* Discord: use /gui-download-url command in any channel
</Step>

<Step title="Launch the installer">
![](/images/Screenshot-2026-02-28-at-15.58.39.png)
</Step>

<Step title="Fill out the form">
<Note>
You can get your **license-key** from dashboard or discord-bot (use **/me** command in any channel)
</Note>

<Note>
If its your first time launching BLOOD - we recommend to not touch the **Working Directory** section. If you are migrating from CLI or want to use both you can simply copy the directory path of blood folder and paste there (the GUI will automatically load tasks, strategies and presets)
</Note>

![](/images/Screenshot-2026-02-28-at-16.02.29.png)

<Warning>
#### **If you have error pop-up during installation do following:**

1\. Download CLI version from Blood dashboard

2\. Navigate to `bloodv3` folder located in `C:/Users/<YourUserProfile>/`

3\. Open `gui` folder and place the downloaded CLI file inside it.<br />

After completing these steps:

* Restart the **GUI**
* Paste your key
* Leave the **Working Directory** field empty

\
All files will be saved in `C:/Users/<YourUserProfile>/bloodv3/gui`
</Warning>
</Step>

<Step title="Press '_**Save and Continue'**_ button">
![](/images/Screenshot-2026-02-28-at-16.05.20.png)
</Step>
</Steps>

<Note>
On your first startup we recommend to jump into [rpc-management.md](/gui/rpc-management)and setup config. This is crucial step to make sure everything runs smooth!
</Note>

## Windows

1. Open bot folder.
2. Click on the path, write `cmd` and press `enter`.
3. Type **EXACT NAME OF BOT FILE** and press **ENTER** (example `BLOOD-3.0.0-49a0b5c-win`)
4. Run needed task file and wait for the crash
5. After crash happened: copy the the full text of crash, paste it to txt file, submit this file + exact bot version + OS name + task file in ticket

_**Example:**_\
BLOOD-3.0.0-abcde3\
Windows\
my\_crash.txt\

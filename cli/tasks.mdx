* For each module the bot automatically creates an **example** file which you can use as a startup to your task creation. For example, copy `solswap_example` and rename it to `solswap_youtaskname`
* Keep in mind the bot can't see & can't run any task with **example** name in it. Those tasks are ment for you to preview the fields that are available for specific module
* Keep in mind every file **requires prefix** with name of the module, on screenshot those are _solswap\__ and _dtf\_._ So your file name must have the following format: module\_customname.csv

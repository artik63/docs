<Warning>
The bot does not monitor positions unless you are using one of the following actions (no RPC requests)\
It is safe to have as many positions as you want, it won't affect the strategies performance
</Warning>

| Name                   | Action                                      |
| ---------------------- | ------------------------------------------- |
| **List Positions**     | Show all your existing positions.           |
| **Watch Positions**    | Monitor current positions value.            |
| **Sell Positions**     | Sell specified percentage of your position. |
| **Hide Positions**     | Deactivate positions to stop monitoring.    |
| **Activate Positions** | Activate position to enable monitoring.     |
| **Delete Positions**   | Remove positions from the bot completely.   |

<Tip>
**To enhance security**, **you must now add your Solana wallets directly through the bot.**

All wallets will be encrypted and you will not be able to access or view the private key afterward.

**Please create your wallets separately using any preferred method, and then import them into BLOOD.**
</Tip>

| Name                    | Action                                                                                                                                                                                                                                                                                                                                        |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **List Wallets**        | Show all your added wallets.                                                                                                                                                                                                                                                                                                                  |
| **Add Wallets**         | Add a new wallet to a new or existing Wallet Group using a private key.                                                                                                                                                                                                                                                                       |
| **Remove Wallet**       | Remove the selected wallet from any Wallet Group.                                                                                                                                                                                                                                                                                             |
| **Remove Wallet Group** | Remove a Wallet Group along with all wallets inside it.                                                                                                                                                                                                                                                                                       |
| **Upgrade Wallet**      | **A mandatory update for your wallets.** You **MUST** complete this action in order to run the solswap module |
